
# Online BookStore - E-Commerce Web Application

A modern, fully-functional e-commerce web application for an online bookstore built with React.js. This project demonstrates a complete shopping experience with product browsing, shopping cart, and checkout functionality.

##  Project Overview

This is a responsive single-page application (SPA) that allows users to browse books, add items to their cart, and complete purchases. The application features a clean, modern UI with a purple color scheme and includes multiple pages for a complete user experience.

##  Features

### Core Functionality
- **Home Page**: Hero slider, featured products, about section, and contact information
- **Books Categories**: Browse all available books with search and category filtering
- **Shopping Cart**: Add/remove items, update quantities, view cart summary
- **Checkout Process**: Complete checkout form with shipping and payment information
- **About Page**: Company information, mission, values, team members, and statistics
- **Contact Page**: Contact form with Google Maps integration

### Technical Features
- **React Router**: Client-side routing for seamless navigation
- **Context API**: Global state management for shopping cart
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI/UX**: Clean design with smooth animations and transitions
- **Form Validation**: Input validation for checkout and contact forms

## Technologies Used

- **React.js** (v19.2.0) - Frontend framework
- **React Router DOM** (v7.9.6) - Client-side routing
- **React Icons** (v5.5.0) - Icon library
- **CSS3** - Styling and animations
- **EmailJS** (v4.4.1) - Email service integration (optional)

## Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Setup Instructions

1. **Clone the repository**
   
   git clone https://github.com/zahraasaleh64/Online-BookStore.git
   cd Online-BookStore
   2. **Install dependencies**
   npm install
   3. **Start the development server**
   npm start
   4. **Open your browser**
   - The app will automatically open at `http://localhost:3000`
   - If it doesn't, manually navigate to the URL
